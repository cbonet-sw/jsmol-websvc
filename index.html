jsmol-websvc : embedding the JSmol viewer in a web page

This project site serves as a location for hosting a JSmol service for visualising crystal structures and molecules. JSmol is a pure Javascript and HTML5 implementation of the original Jmol Java desktop application and browser Java applet. JSmol is more suited to platforms (eg. Android and iOS smartphones) that run browsers which do not implement the Java applet feature. For further information, please see the JSmol documentation and Jmol wiki. There is also a useful example of embedding the JSmol viewer in a webpage hosted at Newcastle University. You might be interested in the demo page within this site at https://cbonet-sw.github.io/jsmol-websvc/demo/ to get you started.
